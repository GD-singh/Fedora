Welcome to Fedora Drupal Page.

The goal of this work is to investigate the possibilities for combining the benefits of these two technologies and for designing a framework of a content management system that supports both

Approach Combine the popular content management system Drupal with a Fedora Commons repository, using Islandora as a middleware This approach would provide a Drupal-based front end enabling an intuitive interface, and rich functionalities for supporting individual users and communities and a Fedora-based backend that enables the use of Semantic Web technologies that would support efficient discoverability, search and sharing of digital resources
